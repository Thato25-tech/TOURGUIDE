# Media Files Required

This directory should contain the following media files for the Lesotho Interactive Tour Guide application:

## Videos
- lesotho_intro.mp4 - Welcome video for the home screen
- thaba_bosiu.mp4 - Video tour of Thaba Bosiu
- sehlabathebe.mp4 - Video tour of Sehlabathebe National Park
- maseru.mp4 - Video tour of Maseru city
- katse_dam.mp4 - Video tour of Katse Dam
- pioneer_mall.mp4 - Video tour of Pioneer Shopping Mall

## Audio Files
- thaba_bosiu_audio.mp3 - Audio narration for Thaba Bosiu
- sehlabathebe_audio.mp3 - Audio narration for Sehlabathebe National Park
- maseru_audio.mp3 - Audio narration for Maseru city
- katse_dam_audio.mp3 - Audio narration for Katse Dam
- pioneer_mall_audio.mp3 - Audio narration for Pioneer Shopping Mall

## Media Specifications

- Videos: MP4 format, recommended resolution 1280x720 or higher
- Audio: MP3 format, recommended bitrate 192kbps or higher

Please note that if these files are missing, the application will still run but will display error messages when attempting to play media.
